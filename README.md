# Template / Sandbox: react-ts-material-ui-rtk-vite-linter

## Includes

React v19 examples:

- `use()` hook.
- `useActionState()` hook.
- `useFormStatus()` hook.

## Development

add in to `.env` file

```bash
VITE_API_URL=https://fakestoreapi.com
```

- `pnpm install`
- `pnpm run dev`

## Testing

- `pnpm run test`

## Linting

- `pnpm run lint`
