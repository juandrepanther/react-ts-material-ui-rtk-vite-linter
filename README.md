## Template Setup

- `pnpm create . --template react-ts`
- `pnpm add -D vitest`
- `pnpm add @mui/material @emotion/react @emotion/styled`
- `pnpm add @reduxjs/toolkit react-redux`
- `pnpm add -D @types/node eslint-plugin-react`

## Development

- `pnpm install`
- `pnpm run dev`
